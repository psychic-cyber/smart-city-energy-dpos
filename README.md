# Smart City Energy Management using DPoS Blockchain

A Scalable Blockchain Framework for Smart City Energy Management.

## Features

- City-wide energy monitoring
- DPoS blockchain simulation
- Fake record detection
- Smart billing
- Solar energy trading
- High consumption alerts
- Energy theft detection
- Fault detection
- Energy prediction
- Real-time dashboard

## Tech Stack

- Python
- Flask
- MongoDB
- Blockchain (DPoS)
- Solidity
- Ganache
- MetaMask
- Power BI
